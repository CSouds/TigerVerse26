# TigerVerse26

This program repeatedly scans work that a user is doing, and points out any errors. There is checking, tutoring and learning mode, where the user can either have errors pointed out, errors pointed out with a reason, or errors pointed out with a solution.

The program uses the Lens Studio kit and integrates an image capture process with base64 encoding to send to GPT 4o to have fast reasoning on the work check.

AI Notice: This project did use AI during development. All concepts and ideas were from Connor Souders and Carson Mensink. Implementation was often changed or guided by a model. AI was largely used for debugging.
